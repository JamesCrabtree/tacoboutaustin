ABOUT TACOBOUTAUSTIN API

Tacoboutaustin API is a service that provides RESTful APIs to retrieve information about hotels, restaurants, and attractions in Austin to developers.

Endpoints

  • /attractions search for all attractions
  • attractions/{id}/ search for attraction with id = {id}
  • /restaurants search for all restaurants
  • /restaurants/{id} search for restaurant with id = {id}
  • /hotels search for all hotels
  • /hotels/{id} search for hotel with id = {id}
  • /categories search for all categories
  • /zipcodes search for all zip codes

Status Codes and Error Messages

In the response JSON file, the field status indicates the status and also error messages of the request:

  • "OK" : every thing is ok
  • "INVALID_ID" : the given id was found in the database
